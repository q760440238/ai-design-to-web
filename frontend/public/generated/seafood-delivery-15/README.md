# 鲜港达 · 15 页海鲜配送 App

生成时间：2026-05-02T14:56:59.094Z

鲜港达是一款面向中国城市家庭、餐厅小店与办公室团购场景的移动端海鲜配送 App。用户可浏览鱼虾蟹贝、冷冻海产、刺身食材和团购套餐，查看产地、捕捞/到港时间、鲜度等级、检测信息与冷链状态；支持加入购物车、预约配送时段、在线支付、订单冷链追踪、会员权益管理，以及企业采购、多人拼单和企业开票等功能。
